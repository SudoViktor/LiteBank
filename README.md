# LiteBank
Високонавантажена, подієво-орієнтована банківська платформа, побудована на мікросервісній архітектурі, що використовує Apache Kafka для асинхронної обробки транзакцій та забезпечує надійну узгодженість даних.

# Структура
Проект LiteBank построєний на мікросервісній архітектурі, кожен з яких запускається у своєму докер контейнері і спілкується із зовнішнім світом завдяки FastApi або Kafka:
- postgresql: Сервіс бази данних. В проекті використовується 4 окремі бази данних:
  1. auth_service_db - база для користувачів банку
  2. accounts_db - база для рахунків користувачів
  3. transactions_db - база для створення ті перегляду історій транзакцій. Але сама по собі не є першоджерелом коштів, тут лише історія та можливість творити транзакці.
  4. ledger_db - база данних з історією транзакцій. Є першоджерелом, саме тут зберігається інформація в кого скільки коштів на основі історії успішних транзакцій.
- kafka: брокер повідомлень для спілкування сервісів між собою. Тут використовуються наступні топіки:
  1. AccountOwners: Коли сервіс accountservice створює новий рахунок ввін повідомляє сервісу transactionservice хто є власником якого рахунку щоб transactionservice міг створювати транзакції лише від імені власника.
  2. CreateTransaction: Сервіс transactionservice створю транзакцію та повідомляє сервісу ledgerservice що її потрібно обробити.
  3. CompleteTransaction: Коли сервіс ledgerservice обробив транзакцію (успішно чи не успішно) він повібомляє transactionservice щоб він в себе обновив статус транзакції.
- authservice: сервіс для реєстрації та авторизації користувачів. Використовує базу auth_service_db. Для авторизації використовуються JWT токени. Завдяки якому можна отримати доступ до кожного сервісу
- accountservice: сервіс для створення рахунків користувачів та карток.
- transactionservice: сервіс в якому можна створювати транзакції та переглядати історію транзакцій та їх статуси
- ledgerservice: сервіс в якому обробляються транзакції. Саме тут визначається в кого який баланс на основі історії всіх транзакцій.
- front: веб сайт банку.
